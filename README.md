# 🌸 Anime World

Jednoduchý web s tématikou anime vytvořený pomocí HTML a CSS.

## 📁 Struktura projektu

Projekt obsahuje dva soubory:

- `index.html` – hlavní HTML soubor strukturu webu
- `design.css` – stylování webu pomocí CSS

## 🔗 Propojení souborů

CSS soubor je připojen v HTML pomocí:

```html
<link rel="stylesheet" href="design.css">
